<h1 align="center"><b>🚀 KRIPIKU — Tempe Chips E-Commerce Platform</b></h1>

---

## 📌 Overview

**KRIPIKU** is a web-based information system and e-commerce platform developed for **UMKM Keripik Tempe Lesti** to digitize and optimize their sales operations.

This system enables customers to browse products, manage purchases, and complete transactions online, while also providing business owners with tools to manage inventory, orders, and product data efficiently.

The platform integrates modern web technologies with real-world services such as shipping cost calculation and online payment gateways, delivering a complete digital commerce solution for small and medium enterprises (SMEs).

---

## 🎯 Key Highlights

- Full-featured e-commerce system for UMKM  
- Integrated shipping cost calculation (Raja Ongkir API)  
- Secure online payment with Midtrans  
- Role-based authentication (Admin & User)  
- Real-time stock validation system  

---

## ✨ Features

- 🔐 Authentication System (Login & Register)  
- 👥 Multi-role Access Control (Admin & User)  
- 🛒 Add to Cart functionality  
- ➖ Update & Remove items from cart  
- 📦 Product stock availability checking  
- 🚚 Shipping cost calculation (Raja Ongkir Integration)  
- 💳 Secure Online Payment (Midtrans Integration)
